命令，或者一个程序 > file<br>
将一个命令或者程序输出的内容，输出到file文件中（不存在则新建），会覆盖原有的内容，>>不会覆盖原有的内容，而是追加到file文件的末尾<br>

命令(程序没有试过) < file<br>
使用file作为命令的输入<br>

n >& m 将输出文件m 和输出文件n 合并<br>
样例 command >file 2>&1 将stdout 和stderr 合并输出到file

n <& m 将输入文件M 和n 合并

<< tag 使用tag标识开始和结束输入，标识结束时，必须顶格 

数字和>或者<之间不能有空格  
0 = stdin  
1 = stdout  
2 = stderr
