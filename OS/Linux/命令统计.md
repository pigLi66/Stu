 

##  快捷键

| 按键     |                    作用                    |
| -------- | :----------------------------------------: |
| Ctrl + d |           键盘输入结束或退出终端           |
| Ctrl + s |   暂停当前程序，暂停后按下任意键回复运行   |
| Ctrl + z | 将当前程序放到后台运行，回复到前台为命令 fg |
| Ctrl + a | 将光标移动到行头 |
| Ctrl + e | 将光标移动到行末 |
| Ctrl + k | 删除从光标所在位置到行末 |
| Alt + Backspace | 向前删除一个单词 |
| Shift + PgUp | 将终端显示向上滚动 |
| Shift + PgOn | 将终端向上滚动 |
